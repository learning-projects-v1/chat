auth:
- Post api/auth/register - {password, email}
- Post api/auth/login - {email, password}

2. Patch api/friends/{id}/accept- {}
3. Get api/friends - {}
4. Get api/friends/requests - {}
5. Patch api/friends/{id}/reject - {}
5. Delete api/friends/:friendId
8. load_messages(sender_id, receiver_id)
9. send_message(sender_id, receiver_id, message)

Message:
- Post api/Messages/send - {chatDto} -- send message
- Get api/Messages/thread/:id -- get messages in a thread
- Post api/messages/read - {dto} - mark read/unread

ChatOverview:
- Get api/ChatOverview -- gets chat previews of each friend

ChatThread:
- Post api/threads - create thread
- Post api/threads/:id/participants - add user
- delete api/threads/:id/participants/:userId - remove user
- get api/threads/:id/participants

Reactions:
- Post api/Reactions/send - {dto} -- send reaction
- Get api/Reactions/thread/:id -- get reactions for a chat-thread

MessageReadStatus:
- Get api/seen_status/message/:id/user/:id
- Post api/seen_status/message/:id/user/:id
