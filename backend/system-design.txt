Functional Requrements:
1. User A can chat with User B.
2. User A can add a user with Id/username B as friend
3. Chat notification
4. Create group for chat
5. Can share image
6. Online status

Non Functional Requirements:
1. High availabilty
2. Encryption
3. Web Sockets
4. Low latency
